## Setting up

1. Grab the repo
```bash
git clone https://github.com/RollForGuild/CharacterGenerator.git
```

2. Install dependencies
```bash
npm install
# or
yarn install
```

3. Run the server
```bash
npm run dev
# or
yarn run dev
```
